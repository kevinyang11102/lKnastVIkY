# 项目简介 java527

本项目是一款商品进销存管理系统，采用前后端分离的技术架构，后端基于 SpringBoot 与 MyBatis，前端采用 Vue 框架。

## 技术栈

### 后端

- SpringBoot
- MyBatis
- MySQL 5.7

### 前端

- Vue

## 功能模块

系统主要包含以下功能模块：

- 用户管理
- 商品管理
- 进货管理
- 销售管理
- 库存管理

## 系统角色

系统内置超级管理员角色，负责系统的日常维护与管理。

## 运行环境

- 开发工具：IntelliJ IDEA
- 运行方式：SpringBoot 直接运行

## 部署步骤

1. 确保MySQL 5.7环境已正确安装并配置。
2. 导入数据库脚本，初始化数据。
3. 使用 IntelliJ IDEA 导入项目。
4. 配置项目相关参数。
5. 运行 SpringBoot 主程序。

- 基于成熟技术栈开发，保障系统稳定性。
